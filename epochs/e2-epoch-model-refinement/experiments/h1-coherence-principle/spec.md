# H1 — Coherence Principle

**Hypothesis:** A thematic area ("all of these touch the same concern") is a sufficient
and more broadly useful grouping principle than a conjunctive proposition. An epoch is
a concern area with a set of experiments; experiments are independent by default.

**How we'll know:** We can write a CLAUDE.md epoch model that covers all grouping cases
encountered so far (E1 was a thematic batch, not a conjunctive proposition) without
needing to classify the logical relationship between experiments upfront.

**Chosen methodology:** method-iteration (design phase only — no implementation)
