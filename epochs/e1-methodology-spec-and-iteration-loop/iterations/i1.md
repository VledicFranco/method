## What Was Learned

- The "Use these exact field names" block is load-bearing guidance. Without it, agents need
  one failed attempt per phase to discover field names from error messages. With it, zero failures.
  All new methodology YAMLs must include this block from the start.

- Soft warning copy issue: the warning says `"optional_note" is required but missing` —
  "required" is misleading for a soft invariant. Small wording fix needed.

- TC prompts were written before the field name fixes and used stale field names in their
  "submit this" instructions. The server correctly blocked them. Test specs need updating
  if ever run formally again.

- `method_status` returns a `completed_phases` array not in the original spec — useful
  addition that appeared naturally in the implementation.
