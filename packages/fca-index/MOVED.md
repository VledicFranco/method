# @methodts/fca-index has moved

This package was migrated to the [Fractal Co-Design](https://github.com/VledicFranco/fractal-co-design) project on 2026-04-25.

- **New package:** `@fractal-co-design/fca-index`
- **New repo:** https://github.com/VledicFranco/fractal-co-design/tree/main/packages/fca-index
- **Install:** `npm install @fractal-co-design/fca-index`

The `@methodts/fca-index@0.4.x` line will continue to work but is no longer being updated. Please migrate to `@fractal-co-design/fca-index@1.0.0+`.

## Why?

`fca-index` is a tool that implements concepts from FCA (Fractal Component Architecture) and ECD (Extreme Co-Design) methodologies. Those methodologies have an open-source home now at `VledicFranco/fractal-co-design` — moving the tool there puts it next to its canonical specs (`canon/`), the Claude Code skill family (`fcd-*`), and future related tools.
