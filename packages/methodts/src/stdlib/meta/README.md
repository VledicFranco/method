# stdlib/meta/ — Meta-Methodology Stdlib Entries

Meta-methodology definitions in the standard library. These are higher-order methodologies that operate on other methodologies (reflection, governance of the method system itself).

| File | Description |
|------|-------------|
| `p0-meta.ts` | P0-META — the root meta-methodology: manages the method system's own evolution |
| `p-gh.ts` | P-GH meta operations — GitHub-specific meta-methodology support |
| `d-meta.ts` | Domain meta: domain theory operations used across methodologies |
