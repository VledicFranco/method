# stdlib/methods/p3gov/ — P3-GOV Method Definitions

Method files for the P3-GOV (Governance) methodology — steering council and decision management.

| File | Method | Description |
|------|--------|-------------|
| `m1-draft.ts` | M1-DRAFT | Draft governance decision or agenda item |
| `m2-review.ts` | M2-REVIEW | Governance review session — council deliberation |
| `m3-approve.ts` | M3-APPROVE | Approval gate — records decision with evidence |
| `m4-handoff.ts` | M4-HANDOFF | Decision handoff — routes approved decisions to execution |
