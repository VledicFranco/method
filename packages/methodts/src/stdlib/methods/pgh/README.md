# stdlib/methods/pgh/ — P-GH Method Definitions

Method files for the P-GH (GitHub) methodology — GitHub-specific operations.

| File | Method | Description |
|------|--------|-------------|
| `m1-triage.ts` | M1-TRIAGE | Issue triage — classify, label, and prioritize incoming issues |
| `m2-review.ts` | M2-REVIEW | Pull request review — structured code review with critique output |
| `m3-resolve.ts` | M3-RESOLVE | Conflict resolution — addresses review comments and rebases |
| `m4-work.ts` | M4-WORK | Issue → branch → implementation → PR workflow |
