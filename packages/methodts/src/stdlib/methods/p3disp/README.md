# stdlib/methods/p3disp/ — P3-DISP Method Definitions

Method files for the P3-DISP (Dispatch) methodology — task decomposition and parallel agent dispatch.

| File | Method | Description |
|------|--------|-------------|
| `m1-interactive.ts` | M1-INTERACTIVE | Interactive decomposition — collaborates with human to break down task |
| `m2-semiauto.ts` | M2-SEMIAUTO | Semi-autonomous dispatch — human approves decomposition, agent executes |
| `m3-fullauto.ts` | M3-FULLAUTO | Fully autonomous dispatch — agent decomposes and executes without approval |
