   Issue:      Are events persisted to disk or in-memory only? Format? Recovery?
   Blocker:    PRD says "genesis-events.yaml or in-memory store" but doesn't choose
   Resolution: Add "Event Log Durability" section to PRD 4.1
   Recommended: In-memory only for Phase 1; persistent YAML for Phase 2 when Genesis runs
   Time:       5 minutes to add

   Time:       5 minutes to add
