# Sustainability — Autonomous Revenue for Method Development

This directory documents the sustainability sub-project: designing and operating autonomous
micro-businesses powered by the Method system to fund ongoing development, research, and
hardware acquisition.

## Objective

Generate recurring revenue sufficient to cover:
1. **Claude API credits** for development, research, and business operations
2. **Hardware** for SLM training and local inference (GPU, storage)
3. **Surplus** for expanding research capacity

## Context

- **Operator:** Franco, Puebla Mexico, registered as Persona Fisica con Actividades Empresariales
- **Tax regime:** RESICO (1-2.5% ISR on revenue up to 3.5M MXN/year, 0% IVA on exports)
- **Infrastructure:** Method bridge (multi-agent orchestration), Pacta SDK, SLM cognitive cycle,
  strategy pipelines, event triggers, MCP tool surface
- **Hardware:** RTX 4090 (24GB VRAM) for local SLM training/inference
